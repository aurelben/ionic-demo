Utilisation de l'application
============================

L'utilisation de notre application est tres basique et se decoupe en 3 onglets :

* Status : page d'accueil
* File list : liste des fichiers deja importe dans l'application
* Upload : liste les fichiers .his du repertoire de telechargement afin de les transferer dans l'application

Uploader un fichier :
---------------------

Rendez vous dans l'onglet `Upload his file`
Ensuite il vous suffir d'appuyer sur le bouton `move <nom_du_ficher>` du fichier voulu afin de le transferer dans l'application

Exploitation des fichiers importes
----------------------------------

Rendez vous dans l'onglet `Files list`, choisissez le fichier desire et appuyez sur `Read <nom_du_fichier>`

Apres un chargement dont le temps depend de la taille du fichier, vous verrez apparaitre un tableau contenant le
recapitulatif pour toutes les donnees souhaitees.
Un appui sur une ligne du tableau (a l'exception de l'entete) fera apparaitre les graphiques highcharts associees
apres un court chargement.

Il se peux que les graphique mettent un peu de temps a s'afficher pendant lequel le scroll ne sera pas possible.

Dans le cas de la ligne "LOCAL_WS_2MIN_MNM" un troisieme graphique sera present sous forme de rose des vents. De plus
3 boutons apparaitrons afin de vous permettre de changer le nombre de directions affichees et ce sans recharger les donnees.
